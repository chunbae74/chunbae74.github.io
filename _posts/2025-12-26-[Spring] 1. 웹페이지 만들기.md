## 1. 간단한 페이지 만들기

> ### @Controller

서버 기능을 만들고 싶거든, 아무 클래스에 `@Controller` 써놓으면 된다.

```java
@Controller
public class BasicController { 

}
```

<br><br>

> ### @GetMapping()

사용자가 날리는 get요청을 처리할 때 사용하는 어노테이션이다.<br>소괄호 안에는 url 주소 넣으면 되며, 함수명은 아무렇게나 지어도 괜찮다.<br>
`@ResponseBody` 어노테이션 함께 사용 시, 
text 그대로 출력한다.

```java
@Controller
public class BasicController {

    @GetMapping("/about")
    @ResponseBody
    public String about() {
        return "코딩애플 강의에서 만들어본 예제 사이트입니다.";
    }   

}
```

<br><br>

#### HTML 파일 반환

text가 아닌, HTML 파일을 넘겨줄 수도 있다.<br>
**src/main/java/resources/static 혹은 templates**에 HTML 파일 생성 후,<br>return으로 파일 이름을 넘겨주면 된다.

```java
@Controller
public class BasicController {
    
    @GetMapping("/about")
    public String about() {
        // about.html 파일 반환
        return "about.html";
    }

}
```

<br><br>

### resources/static vs. resources/templates

**static폴더**<br>
서버에서 별도의 가공 없이 있는 그대로 브라우저에 전달된다.
<br><br>
**templates폴더**<br>
템플릿 엔진(Thymeleaf, FreeMaker, Mustache 등)이 HTML파일에 **데이터를 채워넣은 후**, 새로운 HTML을 만들어 브라우저에 전달한다.<br>HTML 파일 내부에 `th:text="${name}"`과 같은 템플릿 문법을 사용할 수 있다.

<br><br>

## 2. 템플릿 엔진 (Thymeleaf)

> 템플릿 엔진이란?

템플릿 엔진은 미리 정의된 양식에 실제 데이터를 결합하여 최종적인 문서를 만들어주는 소프트웨어 도구이다.<br>템플릿 엔진을 사용하면 단순한 정적 HTML 파일만으로는 해결하기 어려운 문제들을 해결해 준다.

<br><br>

> 사용예제

**/list** 페이지에 접속하면은 상품 목록들을 띄워주는 홈페이지를 만들어보자.<br>먼저, 템플릿 엔진을 사용하지 않고 코드를 작성해보자.

```java:ItemController.java
@Controller
public class ItemController {

    @GetMapping("/list")
    String list(){
        return list.html";
    }

}
```

```html:list.html
<!doctype html>
<html>
<head>
    <meta charset="UTF-8">
    <meta name="viewport"
        content="width=device-width, user-scalable=no, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>
<body>

<div>
    <img>
    <h4>바지</h4>
    <p>7억</p>
</div>
<div>
    <img>
    <h4>셔츠</h4>
    <p>8억</p>
</div>

</body>
</html> 
```

<br><br>

**Thymeleaf 템플릿 엔진** 사용 시, build.gradle에 다음 문장을 추가해 라이브러리를 설치해야 한다.

```cmd
dependencies {
   implementation 'org.springframework.boot:spring-boot-starter-thymeleaf'
}
```

특정 HTML 파일의 내용을 수정하려면,<br>
> 1. 서버 API 함수의 파라미터에 Model model을 넣고<br>
> 2. 함수 내에서 model.addAttribute("name", 전송할데이터) 명령어 작성<br>
> 3. 대상의 HTML 파일 태그에 th:text=${name} 작성<br>


<br>하면 된다.<br>
아래는 Thymeleaf 템플릿 엔진을 사용하여 코드를 수정한 것이다. 

```java:ItemController.java
@Controller
public class ItemController {

    @GetMapping("/list")
    String list(Model model){
        model.addAttribute("title1", "멋진 바지");
        model.addAttribute("price1", 5000);
        model.addAttribute("title2", "멋진 셔츠");
        model.addAttribute("price2", 6000);
        return list.html";
    }

}
```

```html:list.html
<!doctype html>
<html>
<head>
    <meta charset="UTF-8">
    <meta name="viewport"
        content="width=device-width, user-scalable=no, initial-scale=1.0">
    <meta http-equiv="X-UA-Compatible" content="ie=edge">
    <title>Document</title>
</head>
<body>

<div>
    <img>
    <h4 th:text="${title1}">바지</h4>
    <p th:text="${price1}">7억</p>
</div>
<div>
    <img>
    <h4 th:text="${title2}">셔츠</h4>
    <p th:text="${price2}">8억</p>
</div>

</body>
</html> 
```
